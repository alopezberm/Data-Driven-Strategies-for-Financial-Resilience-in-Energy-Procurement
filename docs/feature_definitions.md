# Feature Definitions

## Time Features
- Month, day, weekday
- Seasonality patterns

## Lag Features
- Spot price lags (1, 7, etc.)
- Futures-related lags where relevant

## Rolling Features
- Moving averages
- Volatility measures

## Future Market Features
- Futures prices
- Open interest

## Purpose
Capture:
- trends
- seasonality
- market expectations
- signals useful for both forecasting and downstream decision policies
